Node JS

Node js é um interpretador de código JavaScript que funciona do lado do servidor. Seu objetivo é ajudar programadores na criação de aplicaçães de alta escalabidade (como um servidor web), com códigos capazes de manipular dezenas de milhares de conexões simultâneas, numa única máquina física. O node js é baseado no intrepretador V8. Foi criado por Ryan Dalh em 2009, e seu desnvolvimento é mantido pela empresa Joyent, onde Dalh trabalha.
