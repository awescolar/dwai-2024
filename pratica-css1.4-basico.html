HTML

HTML (abreviação para a expressão inglesa HyperText Language, que signfica Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de paginas na web

CSS

Cascading Style sheets (CSS) é uma linguagem de folhas de estilo utilizadas para definir a apresentação de documentos escrutis en uma linguagem de marcação, como HTML ou XML

JavasScript

JavaScript é uma linguagem de programação interpretada. Foi originalmente implementada como parte dos navegadores web para que scripts pudessem ser executados do lado do cliente e interagissem com o usuário sem a necessidade desre script passar pelo servidor, controlando o navegador, realizando comunicação assíncrona e alterando o conteúdo do documento exibido

Node JS

Node.js é um interpretador do código JavaScript que funciona do lado do servidor. Seu objetivo é ajudar programadores na criação de aplicações de alta escalabilidade (como um servidor web), com códigos capazes de manipular dezenas de milhares de conexões simultaneas, numa única máquina fisica. O Node.js é baseado no intepretador V8. foi criado por Ryan Dahl em 2009, e seu desenvolvimento é mantido pela empresa Joyent, onde Dahl trabalha.

SASS

Quando pensamos produtividade,logo nos perguntamos e pesquisamos como aumenta-la sem grandes impactos. Então os, pré-processadores e frameworks para CSS vem suprir essa necessidade. Com voces, um pouco de Saas

LESS

LESS é uma linguagem de folha de estilos dinamica desenhada por Alexis Sllier. Ela foi influnciada por Sass e influenciou a nova sintaxe "SCSS" do Saas, que adaptou sua sintaxe de formação de blocos do tipo CSS
