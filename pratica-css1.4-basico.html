HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web

CSS

Cascading Style Sheets (CSS) é uma linguagem de folhas de estilo urilizada para definir a apresentação de documentos escritos em uma linguagem de muecação, como HTML, ou XML

JavaScript

JavaScript è una linguagem de programação interpretada. Foi originalmente implementada como parte dos navegadores web para que scripts pudessem ser executados do lado do cliente e interagissem com o umirio sem a necessidade deste script passar pelo servidor, controlando o navegador, realizando comunicação assinerona e alterando o conteúdo do documento exibido.

Node JS

Node.js é um interpretador de código JavaScript que funciona de lado do servidor. Seu objetivo é ajudar programadores na criação de aplicações de alta escalabilidade (como um servidor web), com códigos capazes de manipular dezenas de milhares de conexões simultâness, numa única máquina fisica. O Nodejs é baseado no interpretador V8. Foi criado por Ryan Dahl em 2009, e seu desenvolvimento é mantido pela empresa Joyent, onde Dahl trabalha

LESS

LESS é uma linguagem de folha de estilos dinámica desenhada por Alesis Sellier. Ela foi influenciada por Sass e influenciou a nova sintaxe "SCSS" do Sass, que adaptou son sintaxe de formação de blocos do tipo CSS.
